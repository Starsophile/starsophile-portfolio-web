Agentic IDE Prototype

Problem

Coding assistants are passive. They wait for input rather than anticipating context.

Assumption

Cursor movement and file switching patterns can predict the necessary context window.

Build

A Python-based background service using `watchdog` to monitor file system events and dynamically update a local LLM's system prompt.

Language Python 3.10
Inference Llama 3 (Local)
Latency ~200ms
